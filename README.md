# Proyecto de aprendizaje con NestJS

Este repositorio es un proyecto personal creado con el objetivo de **aprender y practicar NestJS**. No está pensado para uso en producción.

## Descripción

Aplicación base construida con NestJS para experimentar con conceptos como:

* Arquitectura modular
* Controladores y servicios
* Inyección de dependencias
* Manejo de rutas y lógica de negocio

## Instalación

```bash
npm install
```

## Ejecución

```bash
# desarrollo
npm run start

# modo watch
npm run start:dev

# producción
npm run start:prod
```

## Tests

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# coverage
npm run test:cov
```

## Notas

Este proyecto puede contener código experimental, cambios frecuentes o implementaciones incompletas, ya que su finalidad es educativa.
